# 🎬 Netflix Data Analysis Dashboard

## 📌 Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, SQLite, Matplotlib, and Power BI. It performs data cleaning, exploratory data analysis (EDA), visualization, database integration, and builds an interactive dashboard.

---

## 🚀 Features

- Data Cleaning using Pandas
- Missing Value Handling
- Duplicate Removal
- SQLite Database Integration
- Exploratory Data Analysis
- Power BI Interactive Dashboard
- Data Visualization using Matplotlib

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- SQLite
- Power BI
- VS Code
